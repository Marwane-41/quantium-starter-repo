Pink Morsel Sales Analysis

This project analyzes sales data for Soul Foods' Pink Morsels and visualizes trends before and after a price increase on January 15, 2021.


Overview : 

- Processed multiple CSV files into a single clean dataset
- Filtered data to include only Pink Morsel transactions
- Calculated total daily sales 
- Built an interactive dashboard using Dash and Plotly

How to Run

- Clone the repository:

  git clone <your-repo-link>
  cd <repo-name>

- Install dependencies:
  pip install 
  
- Run the app:

  python app.py



- Open your browser and go to:

  http://127.0.0.1:8050/


- run Tests : 
  ./run_tests.sh
